# ⚡ FlashPrompt

**快、轻、爽 —— 告别繁琐，让 AI 提示词调用重回“闪电”速度。**

---

## 🧐 痛点背景

你是否也曾厌倦了这样的流程：

1. 切换窗口到 Notion/备忘录。
2. 搜索并找到那条打磨好的 Prompt。
3. 复制 -> 切回 ChatGPT -> 粘贴。
4. **最痛苦的：** 手动移动光标删掉 `[主题]`，填入新内容。

市面上的插件要么**太重**（强制注册、隐私担忧），要么**太傻**（不支持动态变量）。于是，我找 Gemini 合作写了这款 **FlashPrompt** —— 一把专为效率控打造的“瑞士军刀”。

---

## 🚀 核心特性

### 1. 丝滑的 `/` 指令调用

无需动鼠标，在任何网页输入框（ChatGPT, Claude, DeepSeek 等）直接输入 `/` 即可唤起：

* **输入 `/` 呼出菜单**：使用上下键快速选择。
* **智能排序**：自动记忆上次使用的 Prompt，结合收藏夹置顶，最常用的永远在第一位。
* **快速搜索**：支持搜索标题和标签。
![FlashPrompt-调用1](https://github.com/user-attachments/assets/5f387809-9594-4760-826a-0bd0f7dca257)


### 2. 强大的“动态变量”功能（杀手锏）

告别手动改词。支持使用 `{{}}` 定义变量，调用时自动弹出填空窗口：

> **示例：**
> `请作为一名{{职业}}，用{{风格}}的语气，写一篇关于{{主题}}的文章。`

* **自动化填空**：按 `{{值1}}{{值2}}{{值3}}` 顺序输入即可一键生成。
* ![FlashPrompt-动态变量](https://github.com/user-attachments/assets/8ae35f8b-804d-4c72-af16-15268b243784)

* **进度提示**：右下角实时显示已填变量计数（如 `已填 0/3`），全部填完变绿，治愈强迫症。
* <img width="1920" height="1080" alt="FlashPrompt-窗口细节" src="https://github.com/user-attachments/assets/b4dd9a01-1971-4dc9-a689-5012dbfab84c" />


### 3. 极致轻量与隐私保障

* **无需注册**：没有后台，没有登录。
* **纯本地运行**：数据存储在浏览器的 `Local Storage` 中。
* **零延迟**：没有网络请求，秒开秒用。
* **绝对安全**：你的独家 Prompt 永远留在你的电脑里。

## ⚖️ 开发者交底

* **优点**：极度轻量、无广告、排版还原预设、无多余字符、隐私安全。
* **缺点**：纯粹的工具，**不提供**内置的提示词库。你需要配置适合你自己的“武器库”。

### 💡 如何安装？

1. 下载本项目代码或从 [可疑链接已删除] 下载打包好的文件。
2. 打开 Chrome/Edge 扩展程序页面 (`chrome://extensions/`)。
3. 开启“开发者模式”。
4. 点击“加载解压的扩展程序”，选择本项目文件夹即可。
